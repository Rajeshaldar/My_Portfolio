Home Automation with AR & IoT

Under Prof. Nilanjan Mukhopadhyay's guidance, I developed this project for my 3rd-year mini-project in 2023.

The system utilizes the ESP32-WROOM-32 microcontroller board, enabling wireless control via the ‘Blynk IoT’ app and the internet. Evaluation criteria include functionality, usability, and user satisfaction

A UWB slotted Microstrip Patch Antenna for 5G application

Under Prof. Dipankar Saha's guidance developed this project for 4th year project stage-I in 2023.

This project is to design and analysis the Microstrip Patch Antenna which covers the Ultra-Wide Band in range of 35.492 to 26.649 GHz. This antenna shows a bandwidth of 8.855 GHz, a return loss of -30.002 dB at resonant frequency of 27.838 GHz, and acceptable gain of 5.247 dBi.
